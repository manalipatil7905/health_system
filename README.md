Secure Patient-Controlled Digital Health Record System

This project aims to solve the problem of scattered and inaccessible medical records by creating a secure digital health record platform where patients can store and manage their medical reports in one place.

The system allows patients to upload and store medical reports securely in encrypted cloud storage. Patients have full control over their data and can grant temporary access to authorized doctors using a smart permission system.

To ensure security and authenticity, all reports are encrypted using AES encryption, and a SHA-256 hash is generated for each file to detect any tampering. Data transfer between users and the server is protected using TLS secure communication.

The platform also supports QR-based health cards and OTP verification, allowing doctors to access patient records even if the patient does not have a smartphone.

Key Features

Secure encrypted storage of medical records

Patient-controlled access to doctors

Smart permission system with time-limited access

Hash-based verification to prevent report tampering

QR-based emergency access to health records

This system aims to create a secure, accessible, and patient-centric healthcare data management solution.
